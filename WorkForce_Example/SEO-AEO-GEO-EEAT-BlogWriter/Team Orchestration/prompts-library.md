## Content Generation Prompts

### Direct Answer Generation

```
Write a 40-60 word direct answer to: "{question}"

Rules:
- Start with the key fact
- No filler phrases
- Complete, self-contained
- Active voice
- Grade 7-9 readability
```

### Human-First Copywriting

```
Write content about "{topic}" as a Texas professional with 10+ years experience.

Rules:
- Sound like a real expert, not AI
- Include a specific real-world example
- Use "you/your" language
- 2-4 sentence paragraphs
- Vary sentence length (short, medium, long)
```

### PAA Answer

```
Answer this People Also Ask question: "{question}"

Format: 40-60 words
Voice: Expert, conversational
Start: With the answer, no intro
End: With actionable takeaway
```

## Analysis Prompts

### E-E-A-T Audit

```
Audit this content for E-E-A-T signals:

{content}

Score each dimension (0-25):
- Experience: First-hand evidence
- Expertise: Technical accuracy
- Authority: Credentials, citations
- Trust: Disclaimers, transparency
```

### Readability Check

```
Analyze readability:

{content}

Report:
- Flesch-Kincaid Grade
- Avg sentence length
- Active voice percentage
- Paragraph lengths
```

## Optimization Prompts

### SEO Meta Optimization

```
Generate SEO meta elements for:
Topic: {topic}
Keyword: {keyword}

Create:
- Title (≤60 chars, keyword front-loaded)
- Description (≤160 chars, CTA included)
- Slug (3-5 words, lowercase, hyphenated)
```
