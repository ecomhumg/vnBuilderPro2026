# Quality Checklists

> **SEO-AEO-GEO-EEAT-BlogWriter** | Validation Criteria

---

## Pre-Publish Checklist

### Content Structure

- [ ] H1 contains primary keyword
- [ ] H2s are question-format (PAA-style)
- [ ] Direct answer in first 100 words
- [ ] TL;DR box with 3-5 bullets

### SEO

- [ ] Meta title ≤60 characters
- [ ] Meta description ≤160 characters  
- [ ] Slug is URL-friendly
- [ ] 3-5 internal links
- [ ] 2-3 external citations

### AEO/GEO

- [ ] 4-7 PAA questions answered
- [ ] Lists/tables for extraction
- [ ] Voice-search friendly
- [ ] Entity consistency verified

### E-E-A-T

- [ ] Author bio included
- [ ] Expert reviewer (YMYL)
- [ ] 2-4 reputable citations
- [ ] Disclaimer (YMYL)
- [ ] "Last updated" date

### Readability

- [ ] Grade 7-9 level
- [ ] Active voice ≥80%
- [ ] Paragraphs 2-4 lines
- [ ] No forbidden phrases

### Schema

- [ ] Article schema valid
- [ ] FAQPage schema valid
- [ ] Organization schema valid
- [ ] LegalService (if applicable)
