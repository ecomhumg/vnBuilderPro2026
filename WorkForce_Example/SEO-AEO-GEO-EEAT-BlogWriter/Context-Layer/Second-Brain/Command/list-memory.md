# list-memory - List Memory Entries

> **CLAWDBOT v2.3.2** | Command | Memory Operations

---

## Usage

```
/list-memory [category]
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| category | Optional | Filter by category (seo, aeo, geo, eeat) |

## Purpose

List all entries in MEMORY.md, optionally filtered by category.

## Output

```
Memory Entries [category]:
1. [Entry title] - [Date added]
2. [Entry title] - [Date added]
...
Total: [X] entries
```
