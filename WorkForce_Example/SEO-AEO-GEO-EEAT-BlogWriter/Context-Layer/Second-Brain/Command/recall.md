# recall - Quick Memory Retrieval

> **CLAWDBOT v2.3.2** | Command | Memory Operations

---

## Usage

```
/recall <topic>
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| topic | Yes | Topic to recall |

## Purpose

Quick retrieval of the most relevant memory for a topic.

## Behavior

1. Search MEMORY.md for topic
2. Return top match with context
3. Include related keys for exploration

## Output

```
Recall: [topic]
---
[Most relevant memory content]
---
Related:
- [related.key.1]
- [related.key.2]
```
