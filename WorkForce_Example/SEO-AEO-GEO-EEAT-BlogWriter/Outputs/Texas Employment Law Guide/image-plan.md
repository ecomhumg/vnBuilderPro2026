## Technical Specifications

### Image Optimization

| Format | Use Case | Compression |
|--------|----------|-------------|
| JPG | Photos, hero images | 80% quality |
| PNG | Infographics, flowcharts | Lossless with optimization |
| WebP | Fallback for all images | 75% quality |
| SVG | Icons, simple graphics | When applicable |

### Responsive Sizes

| Breakpoint | Hero | Secondary |
|------------|------|-----------|
| Desktop (1200px+) | 1200 x 628 | 800 x 600 |
| Tablet (768-1199px) | 900 x 471 | 600 x 450 |
| Mobile (<768px) | 600 x 314 | 400 x 300 |

### Accessibility Requirements

- All images must have descriptive alt text
- Alt text should convey meaning, not just describe visuals
- Decorative images use empty alt=""
- Color contrast meets WCAG 2.1 AA standards
- Images should not be the only way to convey information
