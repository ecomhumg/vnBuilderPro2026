# research-commands - Research Commands Suite

> **CLAWDBOT v2.3.2** | Command | Research Operations

---

## Commands

### /research-keyword

```
/research-keyword <keyword> [--volume] [--difficulty] [--intent]
```

Research keyword metrics and related terms.

### /research-competitor

```
/research-competitor <url> [--depth basic|full]
```

Analyze competitor content structure and gaps.

### /research-paa

```
/research-paa <query> [--limit <n>]
```

Extract People Also Ask questions for a query.

### /research-serp

```
/research-serp <keyword>
```

Analyze current SERP features and top results.

### /research-trend

```
/research-trend <topic> [--period 30d|90d|1y]
```

Check search trend data for a topic.

## Output Format

All research commands output structured YAML for further processing.
