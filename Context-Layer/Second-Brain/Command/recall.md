# recall

> **Command**: Quick memory retrieval
> **Syntax**: `recall <topic> [--recent]`

---

## Description

Quickly retrieve relevant memory for a topic. Optimized for speed.

## Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `topic` | required | Topic to recall |
| `--recent` | false | Prioritize recent entries |

## Examples

```bash
# Quick recall
recall "{topic}"

# Recall recent
recall "{topic}" --recent
```
