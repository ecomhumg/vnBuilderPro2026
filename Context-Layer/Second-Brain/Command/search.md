# search

> **Command**: Search across results
> **Syntax**: `search "<query>" [--scope <results|memory|all>]`

---

## Description

General search across results and memory.

## Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `query` | required | Search query |
| `--scope` | `all` | Search scope |

## Examples

```bash
search "{query}" --scope results
search "{query}" --scope all
```
