# save-all

> **Command**: Save current session
> **Syntax**: `save-all [--name "<name>"]`

---

## Description

Save all current session data to memory.

## Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `--name` | auto | Custom session name |

## Examples

```bash
# Auto save
save-all

# Named save
save-all --name "{Session-Name}"
```
