# research-commands

> **Command**: Research command utilities
> **Syntax**: Various research-related commands

---

## Commands

### Web Research

```bash
/search-web "<query>"        # Web search
/deep-research "<topic>"     # Perplexity research
```

### Academic Research

```bash
/scholar-search "<query>"    # Academic papers
/arxiv-search "<query>"      # arXiv papers
```

### Code Research

```bash
/grep "<pattern>"            # Code search
/find "<filename>"           # File search
```
