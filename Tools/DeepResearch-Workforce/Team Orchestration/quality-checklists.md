# Quality Checklists

> **Verification Standards for Deep Research Outputs**

---

## Tier 1: Query Clarity Checklist

- [ ] Query is unambiguous
- [ ] Scope is defined
- [ ] Objectives are clear
- [ ] Focus areas identified
- [ ] Confidence score ≥ 0.8

---

## Tier 2: Protocol Checklist

- [ ] Methodology selected
- [ ] Sources identified
- [ ] Success criteria defined
- [ ] Timeline established
- [ ] Human approval obtained

---

## Tier 3: Search Coverage Checklist

- [ ] Academic sources searched
- [ ] Technical sources searched
- [ ] Web sources searched
- [ ] Minimum 3 independent sources per claim
- [ ] Efficiency patterns applied

---

## Tier 4: Verification Checklist

- [ ] All key claims identified
- [ ] Primary sources traced
- [ ] Cross-references completed
- [ ] Credibility scores assigned
- [ ] Verification score ≥ 0.95

---

## Tier 5: Extraction Checklist

- [ ] Content structured
- [ ] Citations mapped
- [ ] Evidence graded
- [ ] Data validated
- [ ] Gaps noted

---

## Tier 6: Synthesis Checklist

- [ ] All sources integrated
- [ ] Themes identified
- [ ] Contradictions documented
- [ ] Gaps analyzed
- [ ] Coherence score ≥ 0.85

---

## Tier 7: Quality Checklist

- [ ] All claims have citations
- [ ] No unsupported opinions
- [ ] Logical flow maintained
- [ ] Bias indicators noted
- [ ] E-O score ≥ 0.85

---

## Tier 8: Output Checklist

- [ ] Report structure complete
- [ ] Executive summary present
- [ ] References formatted
- [ ] Cost metrics logged
- [ ] Evolution data captured

---

## E-O Score Calculation

```
E-O = (Coverage × 0.25) + (Verification × 0.30) + 
      (Synthesis × 0.25) + (Clarity × 0.20)
```

**Minimum Passing Score: 0.85**
