# Research Patterns Experience

> **Best Practices from Deep Research Operations**

---

## Effective Query Patterns
- Start broad, then narrow
- Use domain-specific terminology
- Include temporal boundaries
- Specify output expectations

## Source Selection Wisdom
- Always triangulate critical claims
- Prefer primary over secondary sources
- Check source independence
- Note publication dates

## Common Pitfalls
- Over-reliance on single source
- Ignoring contradictory evidence
- Confirmation bias in search
- Missing recent updates
