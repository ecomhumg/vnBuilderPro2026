# Synthesis Patterns Skills

> **Multi-Source Integration Techniques**

---

## Synthesis Approaches
| Approach | When to Use |
|----------|-------------|
| Thematic | Cross-cutting topics |
| Chronological | Historical evolution |
| Comparative | Competing views |
| Hierarchical | Nested concepts |

## Integration Steps
1. Categorize by theme
2. Identify overlaps
3. Note unique insights
4. Resolve contradictions
5. Build narrative

## Conflict Resolution
- Weight by evidence quality
- Consider source credibility
- Note uncertainty
- Present both views
