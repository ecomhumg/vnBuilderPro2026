## Primary Databases
| Database | Focus |
|----------|-------|
| PubMed | Biomedical, life sciences |
| ArXiv | Physics, CS, math preprints |
| Google Scholar | Cross-disciplinary |
| JSTOR | Humanities, social sciences |
| IEEE Xplore | Engineering, technology |

## Search Strategy
1. Boolean operators (AND, OR, NOT)
2. MeSH terms for PubMed
3. Citation tracking (forward/backward)
4. Author search for key researchers

## Quality Assessment
- Peer review status
- Journal impact factor
- Citation count
- Author h-index
