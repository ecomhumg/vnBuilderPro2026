# Claim Verification Skills

> **Fact-Check Implementation**

---

## Verification Steps
1. Extract specific claim
2. Identify original source
3. Cross-reference 3+ sources
4. Check expert consensus
5. Document evidence

## Evidence Grading
| Grade | Criteria |
|-------|----------|
| Strong | Multiple peer-reviewed sources |
| Moderate | Established news + expert |
| Weak | Single source, opinion |
| None | No supporting evidence |

## Red Flag Detection
- Sensational language
- Anonymous sources
- Circular references
- Fabricated quotes
