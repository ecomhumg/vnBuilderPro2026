# Report Formatting Skills

> **Output Generation Standards**

---

## Markdown Standards
- `##` for sections
- `###` for subsections
- `**bold**` for key terms
- `[n]` for citations
- Tables for comparisons

## Citation Format
```
[1] Author. "Title." Source, Date. URL
```

## Report Types
| Type | Word Count | Focus |
|------|------------|-------|
| Brief | 500-1000 | Key points |
| Standard | 1000-3000 | Full analysis |
| Comprehensive | 3000+ | Deep dive |

## Quality Checklist
- [ ] All claims cited
- [ ] Logical structure
- [ ] Executive summary
- [ ] Clear conclusion
