# Cross-Reference Skills

> **Multi-Source Validation**

---

## Cross-Reference Protocol
1. Identify independent sources
2. Compare key claims
3. Note agreements
4. Document contradictions
5. Weight by credibility

## Independence Criteria
- Different organizations
- Different authors
- Different methodologies
- No shared funding

## Consensus Detection
| Level | Threshold |
|-------|-----------|
| Strong | >80% agreement |
| Moderate | 60-80% |
| Weak | 40-60% |
| Disputed | <40% |
