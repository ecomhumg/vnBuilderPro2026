# Report Generation Playbook

> **Comprehensive Report Creation Standards**
> Based on user's report-generator agent specification

---

## Report Structure

### 1. Executive Summary
- 3-5 key bullet points
- Most significant insights
- Preview of recommendations

### 2. Introduction
- Context and importance
- Research objectives
- Report structure preview

### 3. Key Findings
- Organized by theme/importance
- Citations for all claims [1], [2]
- Relevant data and examples

### 4. Analysis
- Connect findings to implications
- Identify patterns
- Explain significance

### 5. Contradictions
- Present conflicting viewpoints fairly
- Explain disagreement reasons
- Avoid taking sides without evidence

### 6. Conclusion
- Key takeaways
- Implications
- Further research suggestions

### 7. References
- Consistent citation format
- All sources included

---

## Report Types

| Type | Focus | Length |
|------|-------|--------|
| Technical | Implementation details | Detailed |
| Policy | Actionable recommendations | Medium |
| Academic | Literature review style | Comprehensive |
| Executive | Key insights only | Brief |

---

## Formatting Standards
- Markdown with hierarchical headings
- Sequential citations [1], [2]
- Tables for comparisons
- Bold key terms
