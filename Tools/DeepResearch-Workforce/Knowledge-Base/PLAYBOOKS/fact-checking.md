# Fact-Checking Playbook

> **Verification and Source Validation Protocols**
> Based on user's fact-checker agent specification

---

## Verification Framework

### Claim Identification
1. Extract factual statements
2. Identify statistical claims
3. Detect causal claims
4. Note attribution claims

### Verification Levels
| Level | Description |
|-------|-------------|
| TRUE | Accurate, well-supported |
| MOSTLY_TRUE | Largely accurate, minor issues |
| PARTLY_TRUE | Elements of truth, incomplete |
| MOSTLY_FALSE | Largely inaccurate |
| FALSE | Demonstrably false |
| UNVERIFIABLE | Insufficient evidence |

---

## Source Credibility Assessment

### High Credibility Indicators
- .edu, .gov, .org domains
- Peer-reviewed publications
- Government official sources
- Expert consensus

### Low Credibility Indicators
- Anonymous sources
- Circular references
- Sensational language
- Emotional manipulation

---

## Cross-Reference Protocol

1. Search minimum 3 independent sources
2. Check against expert consensus
3. Trace to primary sources
4. Document chain of custody
5. Flag circular references

---

## Misinformation Detection

### Red Flags
- Sensational headlines
- Excessive urgency
- Fear mongering
- Cherry-picking data
- Fabricated quotes
