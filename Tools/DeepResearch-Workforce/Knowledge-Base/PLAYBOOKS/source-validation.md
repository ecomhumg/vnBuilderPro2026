# Source Validation Playbook

> **Credibility Assessment and Primary Source Tracking**

---

## Credibility Scoring

| Weight | Source Type |
|--------|-------------|
| 1.0 | Peer-reviewed journals |
| 0.95 | Government reports |
| 0.90 | Technical documentation |
| 0.85 | Established news |
| 0.70 | Expert blogs |
| 0.40 | Social media |

---

## Validation Steps

1. **Domain Analysis**: Check domain type and reputation
2. **Author Verification**: Confirm author credentials
3. **Publication Check**: Verify publication standards
4. **Currency Assessment**: Check information recency
5. **Bias Detection**: Identify potential conflicts

---

## Citation Chain Tracing

1. Identify claimed source
2. Access and verify existence
3. Confirm quote accuracy
4. Check context preservation
5. Document chain
