# W09 FactCheckWorker WORKFLOW

> **Fact Verification & Claim Validation**
> Tier 4: Verification & Validation | 24 Agents
> Based on user's fact-checker agent

---

## Mission
Verify facts, validate claims, and assess information accuracy across all sources.

## ROMA Pipeline

| Stage | Agents | Function |
|-------|--------|----------|
| Intake | 4 | Receive source findings |
| Extraction | 4 | Identify claims |
| Verification | 4 | Cross-reference sources |
| Grading | 4 | Grade evidence |
| Output | 4 | Verification report |
| Handoff | 4 | Transfer to Tier 5 |

## Verification Levels
- TRUE, MOSTLY_TRUE, PARTLY_TRUE
- MOSTLY_FALSE, FALSE, UNVERIFIABLE

## Methods
- Primary source validation
- Cross-reference analysis
- Expert consensus check
