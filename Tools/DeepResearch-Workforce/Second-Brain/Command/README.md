# Research Commands

> **Slash Commands for DeepResearch-Workforce**

---

## Available Commands

| Command | Description | Workers Activated |
|---------|-------------|-------------------|
| `/research <query>` | Execute deep research | All tiers |
| `/clarify <query>` | Analyze query clarity | W01 only |
| `/verify <claim>` | Fact-check a claim | W09-W10 |
| `/synthesize` | Merge current findings | W14-W16 |
| `/report` | Generate final report | W19 |
| `/status` | Check research progress | System |

---

## Usage Examples

```
/research "Impact of AI on healthcare diagnostics 2020-2025"
/verify "GPT-4 achieves 90% accuracy on medical licensing exams"
/clarify "Tell me about machine learning"
/synthesize --format=thematic
/report --style=academic --length=comprehensive
```
