# Quality Checklists

> **ScholarDeepResearch-Workforce v1.0.0**

---

## Search Quality

- [ ] ≥8 academic sources found
- [ ] ≥3 database diversity
- [ ] ≥50% from last 5 years
- [ ] Relevant to query domain

---

## Content Quality

- [ ] Fulltext coverage ≥80%
- [ ] Dataset parsing successful
- [ ] No extraction errors

---

## Critical Reading Quality

- [ ] Summaries ≥50 words each
- [ ] 3 questions per document
- [ ] Assumptions documented
- [ ] Diagrams generated

---

## Citation Quality

- [ ] 100% DOI validity
- [ ] 100% links accessible
- [ ] No hallucinated citations
- [ ] Correct author/year format

---

## Report Quality

- [ ] Overview leads with synthesis
- [ ] ≥8 sources cited
- [ ] Visualizations included
- [ ] Citations properly formatted
