# Patent & Web Search

> **Skill ID**: SKILL-SDR-004
> **Tools**: patentSearch, WebSearchReranked

---

## Overview

Skills for patent literature and non-academic web content discovery.

---

## Tool Usage

### patentSearch

- **Purpose**: Search patent databases
- **Strengths**: IP, technical innovations, prior art
- **Parameters**: query, jurisdiction (optional)

### WebSearchReranked

- **Purpose**: Web search for non-academic content
- **When**: Technical blogs, documentation, news
- **Parameters**: query, max_results

---

## Patent Search Strategies

| Strategy | Use Case |
|----------|----------|
| Keyword | General concept search |
| Classification | IPC/CPC code filtering |
| Assignee | Company-specific search |
| Inventor | Person-specific search |

---

## Web Search Use Cases

- Technical documentation
- Blog posts and tutorials
- Industry news
- Open source projects
- Government reports

---

## Best Practices

1. Use patentSearch for technical innovations
2. Use WebSearchReranked only for non-academic content
3. Verify web sources for credibility
