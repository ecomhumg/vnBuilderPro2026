> **Experience ID**: EXP-SDR-003

---

## Common Hallucination Patterns

| Pattern | Detection | Prevention |
|---------|-----------|------------|
| Fake DOI | Resolution fails | Always verify |
| Non-existent journal | Not in databases | Check journal first |
| Wrong publication year | Metadata mismatch | Cross-reference |
| Fabricated author | Not on paper | Verify authorship |

## Citation Format Template

```
*Author, A. (Year). Title. Journal, Vol(Issue), Pages. [DOI/Link]*
```

## Verification Checklist

- [ ] DOI resolves correctly
- [ ] Link is accessible (200/301)
- [ ] Author names match paper
- [ ] Year is correct
- [ ] Journal exists
