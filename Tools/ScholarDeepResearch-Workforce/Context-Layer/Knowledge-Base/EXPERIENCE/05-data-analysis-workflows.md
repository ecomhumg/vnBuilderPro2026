## Common Workflows

### Dataset Processing

1. Load with pandas
2. Inspect structure
3. Clean missing values
4. Transform as needed

### Statistical Analysis

1. Descriptive statistics first
2. Visualize distributions
3. Apply appropriate tests
4. Report with effect sizes

### Visualization Pipeline

1. Define dimensions
2. Select chart type
3. Apply styling
4. Export high-res
