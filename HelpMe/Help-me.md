## üîç Troubleshooting

### Issue: Workforce generation incomplete

**Solution**: Resume from last checkpoint

```bash
/vnbp-orchestrate --resume
```

Check `PhaseOutputs/` for the latest session folder.

---

### Issue: Quality gate failure

**Solution**: Review the blocking gate

| Gate | Location | Fix |
|------|----------|-----|
| Requirements | After Tier 2 | Clarify scope |
| Architecture | After Tier 3 | Simplify topology |
| E-O Score | After Tier 7 | Add efficiency patterns |

---

### Issue: Memory bus errors

**Solution**: Check keys in `memory-bus-contract.md`

Common missing keys:

- `architecture.swarm.topology`
- `efficiency.memory.compression`
- `output.workforce.path`

---

### Issue: Pattern not found

**Solution**: Extend pattern library

Edit: `Context-Layer/Knowledge-Base/PLAYBOOKS/pattern-application.md`

---

### Issue: Deep Research timeout

**Solution**: Use granular commands

```bash
# Instead of full research
/vnbp-research "narrow topic focus"
```
